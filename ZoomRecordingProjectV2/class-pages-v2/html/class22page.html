Is the JavaScript engine written in C++?

There are a number of JavaScript engines in wide use, including:

Chrome uses one called V8.
Safari uses one that’s part of Webkit, called JavaScriptCore
Edge uses one called ChakraCore
Mozilla uses one called SpiderMonkey
V8, JavaScriptCore and ChakraCore all seem to be written entirely in C++. SpiderMonkey is also largely C++, but includes some C as well.
